# Moe Kyaw Aung — Tailwind CSS + PWA Portfolio

A premium dark bento-grid portfolio built with **Tailwind CSS**, **vanilla JavaScript**, and a **PWA** setup for GitHub Pages.

## Features
- Dark bento-grid developer layout.
- Animated hero with floating avatar.
- Typing headline effect.
- Scroll reveal animations.
- 3D tilt cards on desktop.
- Filterable app collection.
- Social links section.
- Offline-ready service worker.
- Mobile-friendly and reduced-motion support.

## Tech Stack
- HTML5
- Tailwind CSS
- Vanilla JavaScript
- PWA
- GitHub Pages
- Font Awesome
